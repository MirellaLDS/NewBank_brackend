# NewBank_brackend
O servidor em questão fornecerá informações sobre uma conta bancária vinculada ao usuário proprietário do dispositivo.

> [!IMPORTANT]
> Você precisa ter instalado nodejs

**Rodar projeto**
- Local: yarn dev
- Produção: yarn start


**Public API:** 
> https://newbank-backend.herokuapp.com/
